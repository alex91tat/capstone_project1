    GitHub Link

  • Car Repair Shop Simulation

    GitHub Link

    The Car Repair Shop simulation is a comprehensive software application designed to manage the operations of a car repair system. This project allows users to handle various aspects of a car repair shop, including managing customers, employees and car information, tracking services performed, and updating the status of cars undergoing repair. Includes a secure login page to authenticate users before accessing the system, ensuring privacy and security. The application is built with Object-Oriented Programming principles to ensure modularity, maintainability, and scalability, allowing for future extensions and improvements. The system follows a CRUD architecture, providing users with full control over the data. Users can add, retrieve, modify, and delete different types of data through an intuitive interface. The application also integrates with a database to store and retrieve critical information, ensuring persistent data storage and seamless user experience. Technologies used: Java, Java Swing, PostgreSQL.
